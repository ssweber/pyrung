- Nickname Exclusions Characters / Words Example Error Message from Address Picker
- Cannot be used in Nicknames % , " < >! # $ & ' ( )* + - . / : ;= ? @[ \ ] ^ ` { | } ~"SYS " % , " < > are not allowed to be used in nicknames
- Cannot be used as leading character in Nicknames _(Underscore) _ is not allowed to be used as the first character of nicknames
- These words cannot be used as Nicknames * LOG SUM SIN ASIN RAD COS ACOS SQRT DEG TAN ATAN LN PI MOD AND OR XOR LSH RSH LRO RRO LOG is not allowed to be used as a nickname
- Cannot use nicknames that have already been assigned This nickname is already assigned to Address X001
- * However, nicknames can include these words as a part of nicknames. For instance, users can not use 'LOG' as a nickname, but it is allowed to use 'A LOG' as nickname.
